# 设计模式
23 个设计模式 (GoF)。

## 创建型(Creational Patterns)
* [抽象工厂](./creational/abstract-factory/index.spec.js) 工厂来管理相似的类。这些类的构造函数的参数是相同的。实例根据传入的类的名字和参数，由工厂来生产。将创建不同类型的实例的过程解藕了。
* [工厂方法](./creational/factory-method/index.spec.js) 实例由工厂来生产。将创建实例和参数解藕了。
