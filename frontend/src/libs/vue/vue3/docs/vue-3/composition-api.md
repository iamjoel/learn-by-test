## 表单绑定值
用 v-model 绑定。
```html
<input v-model="count" />
```

## 响应式
reactive更适合定义复杂的数据类型（json/arr）。ref适合定义基本数据类型（可接收基本数据类型和对象）

### ref
在 js 中，设置和取值，要用 ref.value。在 template 中，设置和取值直接用 ref，不要加 value。

### reactive
直接改内部的值。

## Props
声明 & 使用属性。
```js
const props = defineProps({
  name: String
})
```

## emit
声明 & 使用传给父组件的事件。

```js
const emit = defineEmits(['change'])
```

## Slot
组件内 Slot 定义
```html
<slot></slot>
<slot name="a"></slot>
<li v-for="(item, index) in [1, 2, 3]" :key="item">
    <slot :item="item" />
</li>
```

父组件传入 Slot
```html
<SlotDefault>
    <div>slot 内容</div>
</SlotDefault>

<SlotMulti>
    <template v-slot:a>还是往A槽丢</template>
</SlotMulti>

<SlotWithData>
    <template v-slot:default="slotProps">
        <div>item: {{slotProps.item}}</div>
    </template>
</SlotWithData>
```
